<h1>NanoGPT From Scratch</h1>

<p1>This project is a step-by-step implementation of a GPT-style transformer language model built entirely from scratch using Python and PyTorch. The primary goal is to understand how modern Large Language Models (LLMs) work by implementing each component rather than relying on high-level libraries.

The project explores the complete language modelling pipeline, from raw text processing to autoregressive text generation. Each stage is implemented and documented to provide a practical understanding of the underlying concepts and mathematics behind transformer-based architectures. </p1>

<h2>Features</h2>

* Text preprocessing
* Character, word, and sub-word tokenization
* Vocabulary construction
* Embedding layers
* Positional embeddings
* Self-attention and multi-head attention
* Feed-forward neural networks
* Residual connections and layer normalization
* Transformer blocks
* Training with backpropagation and cross-entropy loss
* Autoregressive next-token text generation

<h2>Technologies</h2>

* Python
* PyTorch
* NumPy
* Matplotlib (for visualisations where applicable)

<h2>Project Goal</h2>

<p2>The purpose of this project is educational rather than achieving state-of-the-art performance. By building each component from first principles, the project aims to develop a thorough understanding of transformer architectures and the techniques used in modern language models such as GPT.

As development progresses, each component will be added incrementally, tested independently, and integrated into a complete language model capable of generating coherent text.</p2>
